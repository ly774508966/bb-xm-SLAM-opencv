Introduction

Hi.
Welcome to our project page. We are Shashank and Aditya.
We completed this project successfully during our five month long intership. It is a very simple implementation of an indoor navigation system in which a camera is used to study and extract features from landmarks (which we called "markers"). These extracted features help us calculate the distance from the landmark. Hence, if the position of the landmark is known in the given environment, we can calculate the position of our camera.
